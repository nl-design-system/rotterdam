Form Field Date Picker component

Biedt de mogelijkheid om een datum en tijd te selecteren met label, beschrijving en foutmeldingen.

Deze component gebruikt het <rods-date-picker> web component element.

Form Field Date Picker

Een complete form field met label, beschrijving en date picker input.

Form Field Date Picker Required

Een verplichte date picker form field.

Form Field Date Picker Disabled

Een form field in disabled mode. Wordt zo gerenderd door Wicket setEnabled(false) te gebruiken.

Form Field Date Picker without Description

Een form field zonder beschrijving, alleen met label.

Form Field Date Picker with Available Times

Een form field met beschikbare tijdslots geconfigureerd via withDatePicker() methode.

Demonstreert het gebruik van de fluent API om de date picker te configureren.
