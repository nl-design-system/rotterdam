Form Field Label component

Toont een (tekst)label dat informeert over de in te vullen informatie.

Form Field Label - statische inhoud

  • Plaats het label voor het invoerveld, behalve voor selectievakjes en keuzerondjes.
  • Plaats het selectievakje of keuzerondje in het label.
  • Koppel het label met het invoerveld met "for" en "id".
