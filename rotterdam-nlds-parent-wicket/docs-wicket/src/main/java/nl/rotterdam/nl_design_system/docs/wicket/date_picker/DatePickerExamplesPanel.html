Date Picker component

Een date picker component om een datum en tijd te selecteren.

Deze component gebruikt het <rods-date-picker> web component element.

Date Picker With Available Date-times (Desktop)

Een date picker met beschikbare datum-tijden voorgedefinieerd.

Er zijn beschikbare tijden op 15 en 16 januari 2026.

Date Picker With Available Date-times (Mobile)

Een date picker met beschikbare datum-tijden voorgedefinieerd.

Er zijn beschikbare tijden op 15 en 16 januari 2026.

Date Picker With 90 days of office hours available date-times

Een date picker met 90 dagen aan beschikbare datum-tijden tijdens kantooruren.

Date Picker

Een eenvoudige date picker met een ingevulde waarde.

Date Picker Empty

Een date picker zonder ingevulde waarde.

Date Picker Disabled

Een date picker in disabled mode. Wordt zo gerenderd door Wicket setEnabled(false) te gebruiken.

Date Picker Required

Een date picker die verplicht is in te vullen.
