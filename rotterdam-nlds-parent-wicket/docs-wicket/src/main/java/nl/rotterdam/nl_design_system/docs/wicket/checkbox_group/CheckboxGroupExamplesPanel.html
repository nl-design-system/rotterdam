Checkbox group component

Een groep van selectievakjes.

Geselecteerde items hebben op rij-niveau andere css classes.

Refresh met Ajax wanneer de selectie wijzigt, als de style wijzigt bij de selectie.

Kenmerken:

  • Zet het <label> om de checkbox heen
  • Geef het label het NL Design System gedrag via RdFormFieldLabelBehavior
  • Geef de Check het NL Design System gedrag via RdCheckboxBehavior
