Here is a list of files that you want to add if you're making a css component

add a readme file in package/component-css/component-name/
add a css-utrecht-component.stories.tsx file in packages/storybook/src/
add a component.md file in packages/storybook/src/
add or fill the component.tokens.json in proprietary/design-tokens/src/component/utrecht/
if the component needs custom css then add index.scss or css in package/component-css/component-name/
